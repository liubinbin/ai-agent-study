# 小助理敏感操作学会了向上请示

随着处理的事情越来越多，越来越杂，小助理有时候会遇到拿不准的事情，这时候老员工就教他要向上级反馈，拿到上级的意见，再处理。这件事情在平常的职场中，还是挺常见的，在 deerflow 也有实现，就是 ClarificationMiddleware。
deerflow 的实现有两层，是在 make_lead_agent 时，一是 tool 列表增加一个 ask_clarification，二是 middleware 列表增加 ClarificationMiddleware。

## ask_clarification 工具
在 prompt 中增加让 LLM 在如下几种情况下增加信息确认步骤，包括四类：缺失信息、 要求有歧义、方案需要选择、有风险的操作、建议。

## ClarificationMiddleware 插件
在提交 tool 执行前，若发现 tool 名称为 ask_clarification，则暂停处理请求，拼接确认消息返回给客户端。
