# 前置知识

前置知识主要包含四个部分，一、LangGraph 是 deerflow 项目的底座，deerflow 是基于 LangGraph 而设计和开发。二、skill 是目前比较流行，做能力扩展的，也是当前我比较关注的，其他的类似 mcp 暂时关注不多。三、docker 是可以方便的创建一个隔离的执行环境。四、其他

## LangGraph
有关 LangGraph agent 的使用详见[1]，在本章节选取部分重要的内容做解释。
create_agent：此方法主要用来创建 agent。
AgentState：agent 实力的执行状态，可以认为是 agent 的上下文。
中间件函数：before_model、after_model、wrap_model_call 和 wrap_tool_call 等中间件函数可以在 agent 执行过程中插入一些自定义的逻辑。

## Skill
skill 是一个可重复使用的能力，可给 agent 提供特定的工作流和领域知识处理能力。skill 分位两个部分，一个部分是简短的名字和使用说明，可供 agent 在生成计划时填入 prompt，另一个部分是具体的执行的内容，这部分内容会在 agent 决定使用这个 skill 后加载并使用这个 skill 的能力，具体解释详见[2]。

## docker
docker 可以用来生成一个隔离的环境，在 docker 内，可以执行 agent 想要执行的各种任务，而不影响宿主机上的文件，这种情况下，可保证宿主机环境的安全。
