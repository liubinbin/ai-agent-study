# 多智能体协作

QwenPaw 中，你可以创建多个智能体，每个智能体拥有独立的配置、记忆、技能和对话历史。

## multi_agent_collaboration.skill

在 QwenPaw 的自建 skill 中，有一个 multi_agent_collaboration.skill。该 skill 的描述是：当需要其他 agent 的专长、上下文或协作支持，或用户明确要求调用其他 agent 时，使用本 skill。先查询可用 agents，再通过 qwenpaw agents chat 进行双向沟通。

## QwenPaw 对 skill 的设计

QwenPaw 的 skill 分为两层：技能池和工作区技能。

技能池是内置技能和可复用共享技能的来源仓库。

工作区不会直接运行技能池里的条目；要使用技能池中的某个技能，必须先将其广播到工作区。

因此，要开启多智能体协作，只需将 multi_agent_collaboration.skill 从技能池加入到工作区内。

## 如何运作

在多智能体协作时，主要包含三个步骤：

1. 判断是否需要使用其他智能体；
2. 选择哪个智能体；
3. 调用其他智能体。

具体说明：

1. 判断：从 skill 的 md 文件可以看出，如何使用其他智能体是交给大模型去判断的。
2. 选择：可以通过 `qwenpaw agents list` 命令获取所有智能体。
3. 执行：通过 `qwenpaw agents chat` 命令可以与其他智能体沟通，将请求发送给其他智能体，然后获取结果。
